# 抖音评论生成器 Chrome插件

这是一个Chrome浏览器插件，可以为抖音视频自动生成智能评论。

## 功能特点

- 一键生成抖音视频评论
- 智能分析视频标题和内容
- 生成有趣、吸引人的评论
- 简单易用的浏览器插件界面

## 安装说明

1. 打开Chrome浏览器，进入扩展程序页面（chrome://extensions/）
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本插件目录

## 使用方法

1. 打开抖音视频页面
2. 点击Chrome工具栏中的插件图标
3. 在弹出窗口中点击"生成评论"
4. 等待评论生成完成

## 注意事项

- 仅支持抖音网页版
- 请确保在视频页面使用插件
- 评论生成可能需要几秒钟时间
